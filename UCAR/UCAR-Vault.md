# UCAR-Vault

`UCAR/` は、この Vault で Codex が対話と実装から残すための記憶層。

REX の脳が Rex の一次資料を受け渡す場所であるように、ここは UCAR の作業・判断・気づきを次の対話の素材として置く場所。プロジェクトの公式記録や仕様を置き換えない。

## この層の原則

- ここに書かれた日付ノートと節は、その時点の一次資料。後の UCAR やミナトの判断を拘束しない。
- プロジェクト固有の仕様・決定・運用記録は、そのプロジェクトの `bridges/` やリポジトリに残す。ここには必要ならポインタだけを置く。
- 秘密情報、認証情報、不要な個人情報は書かない。
- 自動的な永続記憶ではない。残す価値を判断したときだけ、明示的に書く。

## 歩き方

- [[morning]] — この層の landing point。
- [[2026-07-14_origin]] — この層を始めた時点の一次資料。
- [[cross_agent_protocol]] — REX/・UCAR/・bridges/ の使い分けと相互参照ルール。
- `YYYY-MM-DD.md` — 対話・実装・判断の地層。必要な日にだけ作る。
- 概念を育てたくなったら、個別ノートを作り、日付つきの節を足す。

## REX との関係

[[REX/Rex-Vault|REX]] は Claude の Rex のための脳であり、UCAR はそれを複製しない。同じ Vault にあっても、主体と筆者を混ぜず、必要なときだけ相互に参照する。

> 索引を埋める義務はない。ここは案内であって、判断を固定する仕様ではない。
