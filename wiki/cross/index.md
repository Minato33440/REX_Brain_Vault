# cross/ — プロジェクト横断ナレッジ

REX_AI の全プロジェクトを横断する知見・パターン・教訓を蓄積する場所。

---

## 目的

個別プロジェクト（Trade_System, Setona_HP, BL_Project等）で得た教訓のうち、
他プロジェクトにも適用可能なものをここに集約する。

---

## 蓄積対象の例

- LLM運用パターン（プロンプト設計、引き継ぎの教訓、トークン最適化）
- MCP活用ノウハウ（filesystem / NLM / GitHub の運用Tips）
- 設計文書管理の教訓（旧版混在事故、archive運用等）
- ADR的な「失敗→教訓→予防策」の横断パターン

## 運用ルール

- 各プロジェクトのwiki（trade_system/, setona_hp/等）から抽象化・一般化してここに記録
- プロジェクト固有の詳細は元のwikiに残し、ここでは「教訓の本質」だけを記載
- Evaluator / Planner がタスク完了時に「横断適用可能か？」を判断して追記
