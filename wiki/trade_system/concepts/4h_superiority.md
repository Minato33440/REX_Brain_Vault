# 4H構造優位性フィルター

## 定義

エントリー条件: `neck_4h >= neck_1h`

4H構造（上位足）が1H構造（下位足）を支配していることの確認。
この条件が成立しない場合、半値決済ターゲット（neck_4h）に
構造的な余白がなく、エントリー直後に建値撤退になる。

## 数学的根拠

```
#020検証: sl_4h ≈ sl_1h（100%一致・距離0pips）
共通底slを使うと:
  (neck_4h - sl) >= (neck_1h - sl)
  ↓
  neck_4h >= neck_1h
```

ADR F-1（トップダウン原則）の直接応用。追加パラメータゼロ。

## #026d結果

```
#026c（フィルターなし）: 13件 / PF 2.42 / +113.3p
#026d（フィルターあり）: 10件 / PF 4.54 / +150.6p

除外3件: 全て負けトレード（-13.10, -20.55, -3.70）
勝ちトレード誤除外: 0件
フィルター精度: 100%
```

## 除外される構造パターン

4H neckブレイク後の裏確認パターン。4H SH（neck_4h）が
既にブレイクされた水準にあり、1H SHの方が高い = MTF逆転状態。
